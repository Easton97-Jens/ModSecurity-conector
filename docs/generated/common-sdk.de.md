# Generierter Überblick zum Common SDK

Connector-neutrale Common-SDK-Metadaten für Konfigurationsparser, Request/Response-Helfer, Rule-Merge/Error/Event-Helfer, Manifeste, Runtime-Reports, Origin-Governance, Build-Verträge, C++-Wrapper, Limits, Rule-IDs und Log-Sanitizing. Es wird keine Connector-Laufzeitintegration, keine echte libmodsecurity-Bindung, keine Produktionsreife und keine Full-Matrix-Bereitschaft behauptet.
