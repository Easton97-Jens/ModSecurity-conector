# Historische Dokumentation

**Sprache:** [English](README.md) | Deutsch

Dieses Verzeichnis bewahrt abgelöste Repository-Analysen und Issue-Snapshots.
Sie erhalten ihre ursprünglichen Beobachtungen, Daten, Run-Referenzen und
Claim-Grenzen; sie sind keine aktuelle Konfigurations-, Build- oder
Evidence-Anleitung.

Für aktive Anleitungen den [Dokumentationsindex](../README.de.md), die
aktuellen [Connector-Leitfäden](../connectors/README.de.md) und die
[Evidence-Dokumentation](../evidence/README.de.md) verwenden. Neue historische
Einträge gehören nur hierher, wenn die ursprüngliche Einordnung wertvoll ist;
ein Banner mit Status, Ersatz, Datum und Evidenzgrenze hinzufügen.
