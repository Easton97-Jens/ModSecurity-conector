# Historical documentation

**Language:** English | [Deutsch](README.de.md)

This directory retains superseded repository analyses and issue snapshots.
They preserve their original observations, dates, run references, and claim
boundaries; they are not current configuration, build, or evidence guidance.

Use [the documentation index](../README.md), the current
[connector guides](../connectors/README.md), and
[evidence documentation](../evidence/README.md) for active guidance. New
historical records belong here only when retaining their original context is
valuable; add a banner with status, replacement, date, and evidence boundary.
