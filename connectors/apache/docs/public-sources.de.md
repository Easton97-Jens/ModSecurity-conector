# Öffentliche Apache-Quellen

**Sprache:** [English](public-sources.md) | Deutsch

Status: aktuelle Referenzen

Die Repository-eigene Apache-Connector-Quelle befindet sich unter `connectors/apache/`.
Externe Referenzen dienen der Herkunft, dem Vergleich und sind optional
schreibgeschützte Neuerstellungen:

- Upstream-ModSecurity-Apache-Quelle:
  https://github.com/owasp-modsecurity/ModSecurity-apache
- Apache httpd-Quelle: https://github.com/apache/httpd
- Apache-Modul/APXS-Build-Kontext: Apache httpd-Dokumentation

Quellpins und generierte Build-Roots für die Clean-Clone-Laufzeitvalidierung sind vorhanden
dokumentiert in `docs/build/compilers/apache.md` und den Framework-Helfern `ci/lib/common.sh`.
