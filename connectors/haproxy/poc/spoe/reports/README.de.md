# SPOE Platzhalter für PoC-Berichte

**Sprache:** [English](README.md) | Deutsch

Nur Dokumentation

Dieses Verzeichnis ist nur ein Platzhalter für spätere Dokumentation, nicht für
lokale Report-Erzeugung.

In diesem Connector-Repository werden keine Tests gespeichert.

Alle Testdefinitionen, Testausführungen, Runner und generierten Berichte gehören dazu
Easton97-Jens/ModSecurity-test-Framework.

Berichte werden vom zentralen ModSecurity-Test-Framework generiert, nicht von diesem
Connector-Repository.

runtime_verified muss in diesem Repository bis zum externen Framework false bleiben
Nachweise werden erbracht.
