# HAProxy Test Scaffold

Dieses Verzeichnis beschreibt den Testplan für einen zukünftigen HAProxy-
Connector. Es enthält keine ausgeführte Testimplementierung.

## Geplanter Testumfang (noch zu prüfen)

1. Structure tests
   - Erwartete Dateien/Verzeichnisse vorhanden.
2. Config tests
   - Testkonfiguration validierbar.
3. Startup tests
   - HAProxy + Connector-Komponente starten/stopen stabil.
4. Request/intervention tests
   - Nachweisbare Allow/Block-Entscheidungen.
5. Logging tests
   - Relevante Logs werden erzeugt und korreliert.
6. Generated report checks
   - Reports/JSON vollständig und schema-konform.

## Hinweis

Strukturtests sind nur Vorbedingungen und kein Runtime-Kompatibilitätsnachweis.
